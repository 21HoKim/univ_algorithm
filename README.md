## makefile 사용법
동일한 디렉토리에 *.c 파일이 존재해야함.

명령어 : 
`
make 3-2.c
`

만약 해당 코드의 input과 output을 검사하고 싶으면 해당 소스 디렉터리에 ./data 폴더 생성
*.in, *.out 의 입출력 데이터를 파일이름과 똑같이 생성하면 됨.

[소스코드 구조]
```
3week(sort)/
├── data/
│   ├── 3-2.in
│   └── 3-2.out
├── 3-1.c
├── 3-2.c
├── 3-3.c
├── 3-4.c
├── 3-5.c
├── makefile
└── test.c

```

⚠️출력문 마지막 공백이나 개행문자들을 무시하고 토큰만 입력 받아서 검사함. (1 2 3 == 1\n2\n3\n)
